Installationインストール

There are many ways to install Julia. The following sections highlight the recommended method for each of the main supported platforms, and then present alternative ways that might be useful in specialized situations.Juliaをインストールする方法はいくつかあります。以下のセクションでは、主要なサポートプラットフォームごとの推奨方法を強調し、その後、特定の状況で役立つかもしれない代替方法を紹介します。

The current installation recommendation is a solution based on Juliaup. If you installed Julia previously with a method that is not based on Juliaup and want to switch your system to an installation that is based on Juliaup, we recommend that you uninstall all previous Julia versions, ensure that you remove anything Julia related from your PATH variable and then install Julia with one of the methods described below.現在のインストール推奨は、Juliaupに基づくソリューションです。以前にJuliaをJuliaupに基づかない方法でインストールした場合、システムをJuliaupに基づくインストールに切り替えたい場合は、すべての以前のJuliaバージョンをアンインストールし、PATH変数からJuliaに関連するものをすべて削除し、その後、以下に記載された方法のいずれかでJuliaをインストールすることをお勧めします。

WindowsWindows

On Windows Julia can be installed directly from the Windows store here. One can also install exactly the same version by executingWindowsでは、JuliaはWindowsストアから直接こちらでインストールできます。また、同じバージョンを実行することでもインストールできます。

winget install julia -s msstore

in any shell.任意のシェルで。

Mac and LinuxMac と Linux

Julia can be installed on Linux or Mac by executingJulia は Linux または Mac にインストールできます。実行するには

curl -fsSL https://install.julialang.org | sh

in a shell.シェルで。

Command line argumentsコマンドライン引数

One can pass various command line arguments to the Julia installer. The syntax for installer arguments isさまざまなコマンドライン引数を Julia インストーラーに渡すことができます。インストーラー引数の構文は

curl -fsSL https://install.julialang.org | sh -s -- <ARGS>

Here <ARGS> should be replaced with one or more of the following arguments:ここで <ARGS> は次の引数の1つ以上に置き換える必要があります：

  • --yes (or -y): Run the installer in a non-interactive mode. All configuration values use their default or a value supplied as a command line argument.--yes (または -y): インストーラーを非対話モードで実行します。すべての設定値はデフォルトまたはコマンドライン引数として提供された値を使用します。
  • --default-channel=<NAME>: Configure the default Juliaup channel. For example --default-channel lts would install the lts channel and configure it as the default.--default-channel=<NAME>: デフォルトの Juliaup チャンネルを設定します。例えば --default-channel lts は lts チャンネルをインストールし、それをデフォルトとして設定します。
  • --add-to-path=<yes|no>: Configure whether Julia should be added to the PATH environment variable. Valid values are yes (default) and no.--add-to-path=<yes|no>: Julia を PATH 環境変数に追加するかどうかを設定します。有効な値は yes (デフォルト) と no です。
  • --background-selfupdate=<SECONDS>: Configure an optional CRON job that auto-updates Juliaup if <SECONDS> has a value larger than 0. The actual value controls how often the CRON job will run to check for a new Juliaup version in seconds. The default value is 0, i.e. no CRON job will be created.--background-selfupdate=<SECONDS>: <SECONDS>が0より大きい値を持つ場合に、Juliaupを自動更新するオプションのCRONジョブを構成します。実際の値は、CRONジョブが新しいJuliaupバージョンを確認する頻度を秒単位で制御します。デフォルト値は0で、つまりCRONジョブは作成されません。
  • --startup-selfupdate=<MINUTES>: Configure how often Julia will check for new versions of Juliaup when Julia is started. The default is every 1440 minutes.--startup-selfupdate=<MINUTES>: Juliaが起動したときにJuliaupの新しいバージョンを確認する頻度を構成します。デフォルトは1440分ごとです。
  • -p=<PATH> (or --path): Configure where the Julia and Juliaup binaries are installed. The default is ~/.juliaup.-p=<PATH> (または --path): JuliaおよびJuliaupのバイナリがインストールされる場所を構成します。デフォルトは~/.juliaupです。

Alternative installation methods代替インストール方法

Note that we recommend the following methods only if none of the installation methods described above work for your system.上記のインストール方法がシステムで機能しない場合にのみ、以下の方法を推奨します。

Some of the installation methods described below recommend installing a package called juliaup. Note that this nevertheless installs a fully functional Julia system, not just Juliaup.以下に説明するいくつかのインストール方法では、juliaupというパッケージのインストールを推奨しています。これは、Juliaupだけでなく、完全に機能するJuliaシステムをインストールします。

App Installer (Windows)アプリインストーラー (Windows)

If the Windows Store is blocked on a system, we have an alternative MSIX App Installer based setup. To use the App Installer version, download this file and open it by double clicking on it.システムでWindows Storeがブロックされている場合、代替のMSIXアプリインストーラーに基づくセットアップがあります。アプリインストーラー版を使用するには、このファイルをダウンロードし、ダブルクリックして開きます。

MSI Installer (Windows)MSIインストーラー (Windows)

If neither the Windows Store nor the App Installer version work on your Windows system, you can also use a MSI based installer. Note that this installation methods comes with serious limitations and is generally not recommended unless no other method works. For example, there is no automatic update mechanism for Juliaup with this installation method. The 64 bit version of the MSI installer can be downloaded from here and the 32 bit version from here.WindowsシステムでWindows Storeまたはアプリインストーラー版が機能しない場合、MSIベースのインストーラーを使用することもできます。このインストール方法には重大な制限があり、他の方法が機能しない場合を除いて一般的には推奨されません。たとえば、このインストール方法ではJuliaupの自動更新メカニズムはありません。MSIインストーラーの64ビット版はこちらからダウンロードでき、32ビット版はこちらからダウンロードできます。

By default the install will be a per-user install that does not require elevation. You can also do a system install by running the following command from a shell:デフォルトでは、インストールは昇格を必要としないユーザーごとのインストールになります。また、シェルから次のコマンドを実行することでシステムインストールを行うこともできます。

msiexec /i <PATH_TO_JULIA_MSI> ALLUSERS=1

Homebrew (Mac and Linux)Homebrew (MacおよびLinux)

On systems with brew, you can install Julia by runningbrewがインストールされているシステムでは、次のコマンドを実行することでJuliaをインストールできます。

brew install juliaup

in a shell. Note that you will have to update Juliaup with standard brew commands.シェルで実行します。Juliaupは標準のbrewコマンドで更新する必要があることに注意してください。

Arch Linux - AUR (Linux)Arch Linux - AUR (Linux)

On Arch Linux, Juliaup is available in the Arch User Repository (AUR).Arch Linuxでは、JuliaupはArch User Repository (AUR)で利用可能です。

openSUSE Tumbleweed (Linux)openSUSE Tumbleweed (Linux)

On openSUSE Tumbleweed, you can install Julia by runningopenSUSE Tumbleweedでは、次のコマンドを実行することでJuliaをインストールできます。

zypper install juliaup

in a shell with root privileges.ルート権限でシェルを使用して実行します。

cargo (Windows, Mac and Linux)cargo (Windows、MacおよびLinux)

To install Julia via Rust's cargo, run:Rustのcargoを使用してJuliaをインストールするには、次のコマンドを実行します:

cargo install juliaup
